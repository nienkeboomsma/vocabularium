# Vocabularium

## Installation

1. Create a `.env` file:

```sh
cp .env.example .env
```

2. Make sure to replace the placeholder values.

3. Build the containers:

```sh
docker compose up --build -d
```

## Usage

1. Start the containers:

```sh
docker compose up
```

2. Open `http://localhost:4321` in your browser.

3. Stop the containers:

```sh
docker compose down
```
